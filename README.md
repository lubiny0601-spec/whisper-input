# 轻语输入 / Whisper Input

轻语输入是一款 Windows AI 语音输入工具。它把你随口说的话整理成可以直接发送的文字：去掉口头语，补上标点，保留英文术语，也可以把中文语音直接输出成英文。

## 它解决什么问题

- 说中文，得到自然、干净、能直接发给别人的中文文本。
- 说一段凌乱口语，自动整理成清晰的段落或编号列表。
- 说中文夹英文术语，尽量保留 Gemini、DeepSeek、API Key、ASR、LLM 这类词。
- 说中文，输出英文，适合写英文邮件、Issue、PR 描述或海外沟通。
- 在微信、飞书、浏览器、文档、编辑器等当前输入框中直接落字。

## 为什么不是普通语音转文字

普通语音识别只负责“听见你说了什么”。轻语输入多做一步：让 AI 把口语整理成书面表达。

你可以选择：

- 原文：尽量保持原话，只修正断句和标点。
- 轻度润色：去掉“那个、然后、呃”等口头语，让文字自然可发送。
- 清晰结构：把多个点整理成 `1.`、`1.1`、`2.` 这样的格式化结构。
- 正式表达：把口语改成更正式的邮件或工作沟通文本。

## 一个例子

你说：

> 老板，那个项目验收我刚才说错了，不是周二，是周三下午两点。然后麻烦你看一下合同和付款节点。还有测试，这个地方要改一下。

轻度润色：

> 老板，项目验收时间我刚才说错了，不是周二，是周三下午两点。麻烦你看一下合同和付款节点，还有测试，这个地方要改一下。

清晰结构：

> 1. 项目验收时间更正  
> 1.1 不是周二，是周三下午两点。  
> 2. 待办事项  
> 2.1 麻烦你看一下合同和付款节点。  
> 2.2 测试这个地方要改一下。

正式表达：

> 老板您好：  
> 关于项目验收时间，此前信息有误，现更正为周三下午两点。烦请您查阅合同及付款节点。此外，测试部分需要调整。  
> 谢谢。

## 中文说话，英文输出

如果你经常需要写英文邮件、英文 Issue 或英文工作沟通，可以用中文说出意思，让轻语输入输出英文。你不需要先在脑子里翻译成英文再打字。

## 成本

轻语输入使用你自己的云端 ASR 和 LLM API Key。轻量日常使用通常是每月一两元人民币级别，具体费用取决于你选择的模型、使用时长和供应商计费规则。相比 Typeless 这类按月订阅的语音输入工具，它更适合愿意自己配置 API Key、想把成本压低的用户。

## 隐私

- 录音音频会发送到你配置的云端语音识别服务。
- 识别后的文本会发送到你配置的 AI 润色模型。
- 历史记录和词汇表默认保存在本机。
- 你可以在设置中清空历史记录、词汇表和 API 配置。

## 安装与配置

1. 从 Releases 下载 Windows 安装包。
2. 安装并启动轻语输入。
3. 在“设置 - 模型设置”里选择千问或豆包方案。
4. 填入对应 API Key。
5. 按全局快捷键开始说话。

## 它不是什么

轻语输入不是传统 Windows 输入法，不注册系统 IME；它也不是会议记录工具、聊天机器人或 Agent。它只做一件事：把你说的话整理成可用文字，并尽快放到当前输入位置。
