# 轻语输入 / Whisper Input

轻语输入是一款面向 Windows 用户的 AI 语音输入工具。

按下快捷键，说出你想表达的内容，它会把口语整理成可以直接发送、粘贴或提交的文字：去口头语、补标点、保留英文术语、整理编号结构，也可以把中文口述直接输出成英文。

它适合每天需要写微信、飞书、邮件、需求文档、日报、Issue、PR 描述和工作交接的人。

## 为什么需要它

很多语音输入只能把声音变成逐字稿。问题是，真实口述通常是这样的：

> 嗯那个帮我记一下，今天主要有几个事，第一个是把这几次更新推送到 GitHub 并合并到 main，第二个是 README 要改一下，要参考 Openless 的结构，但我们这个产品主要是给职场员工用的，所以要突出正式输入、中文转英文和数字格式化这些能力。

轻语输入希望输出的是可以直接使用的文字：

```text
本次需要完成以下事项：

1. 代码更新
1.1 将近期更新推送至 GitHub。
1.2 合并到 main 分支。

2. README 改写
2.1 参考 Openless 的文档结构和段落组织方式。
2.2 根据轻语输入的实际定位调整内容。

3. 产品重点
3.1 面向职场员工的高频输入场景。
3.2 突出正式表达、中文口述转英文，以及数字格式化输出等能力。
```

你只负责说，它负责整理。

## 核心能力

- **中文语音输入**：面向中文为主、夹杂英文术语的真实工作场景。
- **AI 润色**：默认把口语整理成自然、干净、能直接发送的文字。
- **四种输出风格**：原文、轻度润色、清晰结构、正式表达。
- **中文口述转英文**：用中文说意思，直接生成英文邮件、Issue 或工作沟通文本。
- **正式表达**：把口述内容整理成邮件、请示、问题反馈、工作说明或交接文档。
- **数字与格式整理**：优化金额、编号、时间、列表和中英文混排。
- **词汇表**：保存你常用的人名、产品名、公司名和专业术语。
- **历史记录**：本地保留最近输入，方便复制和回看。
- **插入失败兜底**：无法直接插入当前输入框时，自动复制到剪贴板。

## 四种输出风格

### 原文

只做基础断句和标点修正，尽量保留原始表达。

适合：聊天、临时记录、不希望 AI 过度整理的场景。

### 轻度润色

删除“呃、那个、就是说、然后”等口头语，合并重复词，补标点，但不改变结构。

适合：微信、飞书、日常工作沟通。

### 清晰结构

把多个点整理成 `1.`、`1.1`、`2.` 这样的编号结构。

适合：任务清单、会议要点、需求拆解、问题复盘。

### 正式表达

把口语转换成正式邮件、公文式说明、问题反馈或工作交接文本。

适合：给老板汇报、对客户说明、写交接文档、写正式反馈。

## 中文说话，英文输出

你可以用中文说：

> 帮我写一段英文，说我们已经完成了这次更新，主要修复了语音长文本截断的问题，并且优化了正式表达模式。

输出可以是：

```text
We have completed this update. The main changes include fixing the issue where long voice input could be truncated, and improving the Formal style so that spoken content is converted into a more structured and professional format.
```

不用先在脑子里翻译，也不用一边想英文一边打字。

## 成本

轻语输入使用你自己的云端 ASR 和 LLM API Key。

日常轻量使用通常是每月一两元人民币级别，具体费用取决于你选择的模型、语音时长和服务商计费规则。相比 Typeless 这类订阅制语音输入工具，它更适合愿意自己配置 API Key、希望长期低成本使用的用户。

## 推荐配置

- 语音识别：千问实时 ASR
- AI 润色：千问 / Gemini / 豆包
- 备用方案：豆包流式语音识别

你只需要在设置里选择服务商并填写 API Key，调用路径和模型会自动设置。

## 隐私说明

- 录音音频会发送到你配置的云端语音识别服务。
- 识别后的文本会发送到你配置的 AI 润色模型。
- 历史记录和词汇表默认保存在本机。
- 你可以在设置中清空历史记录、词汇表和 API 配置。

## 安装与使用

1. 从 [Releases](../../releases) 下载 Windows 安装包。
2. 安装并启动轻语输入。
3. 打开“设置 - 模型设置”。
4. 选择千问或豆包方案。
5. 填入对应 API Key。
6. 按全局快捷键开始说话。

## 它不是什么

轻语输入不是传统 Windows 输入法，不注册系统 IME。

它也不是会议记录工具、聊天机器人、RAG 或 Agent。它只做一件事：把你说的话整理成可用文字，并尽快放到当前光标位置。

## 与 Openless 的关系

轻语输入基于 [OpenLess](https://github.com/Open-Less/openless) 改造而来。

感谢 OpenLess 作者和贡献者在桌面语音输入、全局快捷键、录音状态、文本插入和 Tauri 应用基础设施方面打下的基础。轻语输入在此基础上转向 Windows cloud-first 路线，更聚焦中文职场语音输入、正式表达、中文转英文和低成本 API 使用体验。
